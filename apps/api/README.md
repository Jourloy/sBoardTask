## Getting Started

### Prepare

Before launch app you must create `.env` file and add needed variables. Look into `.env.sample` for more info

### Installation

```bash
$ yarn install
```

### Running the app

#### Yarn

```bash
# Developmemt
$ yarn dev

# Production (Not recommend)
$ yarn start
```

#### Docker-compose

```bash
$ docker-compose up -d
```
