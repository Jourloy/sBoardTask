## Getting Started

### Installation

```bash
$ yarn install
```

### Running the app

#### Yarn

```bash
# Developmemt
$ yarn dev

# Production (Not recommend)
$ yarn start
```

#### Docker-compose

```bash
$ docker-compose up -d
```
